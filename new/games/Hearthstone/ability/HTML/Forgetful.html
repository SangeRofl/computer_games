Стратегия

Основное значение эффекта забывчивости состоит в том, что при наличии нескольких доступных целей игрок не может быть уверен, какая цель будет поражена. Это делает его использование менее целесообразным, когда на доске есть вражеские миньоны с высокой атакой (если они не являются желаемыми целями), так как попадание в них нанесет игроку большой урон. Это добавляет дополнительную непредсказуемость при игре против миньонов, таких как Boom Bots . Забывчивый может быть полезен для обхода миньонов с насмешками. Например, с 1 миньоном-насмешкой и 3 миньонами без насмешек на поле, атака Насмешки с помощью забывчивого миньона будет иметь 50% попадания в Насмешку и 12,5% вероятности поражения любой из других целей. Это может быть полезно для удаления защищенных миньонов или даже для нанесения смертельного урона противнику. Например, если на поле находится только 1 миньон-насмешник, то нападение на насмешку с помощью забывчивого миньона или оружия с вероятностью 50% обойдёт насмешку и поразит вражеского героя. В близких играх этот шанс на дополнительный урон может позволить игрокам сократить разрыв. Из-за этой возможности, когда вы пытаетесь не дать противнику с забывчивыми картами поразить вашего героя, находясь за насмешкой, эффективно вызывать на поле как можно больше миньонов. Каждый из них действует, чтобы уменьшить вероятность того, что атака достигнет вашего героя, если атакующий пропустит намеченную цель. Для дальнейшей стратегии и конкретного использования см.
